# Receipt OCR Extension

**Status:** Skeleton (Issue #3 home)

This extension will handle optical character recognition (OCR) of receipts and invoices to extract transaction data.

**Note:** Not yet implemented. When built, this directory must contain a TypeScript `index.ts` file that exports the pi extension factory, along with a `package.json` manifest. Until then, this skeleton extension is inert (not auto-loaded by pi).
